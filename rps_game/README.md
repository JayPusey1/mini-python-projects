What is the goial of this project?

The goal of this project is to make Rock, Paper, Scissors where the program will go against the user in a best of 3.
The program will report:
 - "What their pick is out of Rock, Paper or Scissors"
 - "Compare their answer to the users"
 - "Determin who won the game"

How do I know when thus program is finished?

- The program will genrate a choice of Rock, Paper or Scissors
- The program will correctly identify the winner of the round
- The program finishes when the program or user wins 2 rounds
- The output is clear and correct

Example of Output:

- Congratulations, you won!
- I won, better luck next time!
